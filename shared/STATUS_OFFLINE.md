## 5. 实操遇到的问题

1. **领先线已退役** — EastMoney trends2 API 已停用，含权vs等权偏离无法直接获取。替代方案：用权重+日线手动算等权，或用中证2000 proxy（r=0.916）
2. **中证2000 分钟数据只有 33 天** — 2025-05 才开始采集，无法做分钟级回测
3. **R²=0.96 只有 16 天样本** — `16日数据框架验证` 的 IF 拔河回归不具统计显著性，需 300+ 日重跑验证
4. **等权数据无直接 API** — tushare 无等权指数接口，需要手动从成分股构建
5. **合成基底权重覆盖 <90%** — Pearson r>0.98 但权重覆盖仅 85%，部分成分数据缺失
